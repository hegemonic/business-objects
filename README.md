# Business objects

### a JavaScript object-oriented business layer for Node.js

See tutorials: [http://bo.logikum.hu](http://bo.logikum.hu)  
See class reference: [http://logikum.github.io/business-objects](http://logikum.github.io/business-objects)

### Installation

If you want to use it in your project:

```
$ npm install business-objects
```

### Tests and Documentation

First clone the repo and install the dependencies:

```
$ git clone https://github.com/logikum/business-objects.git
$ cd business-objects
$ npm install
```

Run the tests:

```
$ jasmine-node tests
```

Generate the documentation with DocStrap or baseline template:

```
$ grunt docstrap
$ grunt baseline
```
